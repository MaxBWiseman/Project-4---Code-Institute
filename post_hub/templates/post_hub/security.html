Security

At Tooter, we take security very seriously. We have implemented the following security measures to protect your data:

  • Trusted AllAuth for authentication, a trusted and widely used authentication system.
  • Several password validators to enforce strong passwords
  • Cross-site request forgery (CSRF) protection
  • Cross-site scripting (XSS) protection
  • Encrypted passwords
  • Cloudinary secure image storage

If you have any security concerns or would like to report a security issue, please contact us at Contact Us.
